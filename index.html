NOTAS

El html es el medio de creacion de paginas por el cual se diseña cualquier pagina que usamos y es un sistema practico que antes hacia paginas plasmadas de informacion bastantes pesadas para la lectura pero gracias a herramientas externas hoy podemos encontrarlas mucho mas accesibles e interactivas para el usuario.


El hipertexto es un medio muy practico a la hora de utilizarlo e las paginas por que evita que el usuario se llene de informacion necesaria permitiendo la facilidad de lectura, la busqueda de informacion y permite un mayor rendimiento en el usuario.

Tim Bernes Lee se destaco por ser el que presento la idea de crear la world wide web el cual es el medio de navegacion mas utilizado en el mundo y se utiliza en la actualidad la que permitio una mayor comunicacion por medio de la internet.

Atributos HTML

Atributos Basicos

  • P-parrafos
  • h1 al h6-titulos de diferente tamaño donde h1 es el mas grande y h6 el mas chico
  • br-salto de un espacio de linea
  • hr-separar por linea
  • ul-lista desordenada
  • ol-lista ordenada
  • li-listado interno a las listas ordenadas y desordenadas
  • b-letras en negritas
  • strong-texto en negritas
  • i-letras en cursiva
  • mark-marcatextos
  • del-texto tachado
  • ins-texto subrayado
  • lorem-atributo de texto falso
  • em-texto en italicas
  • small-texto pequeño
  • sup-texto chico superior

Atributos avanzados

  • img-permite introducir una imagen
  • width-permite ajustar el tamaño del ancho
  • height-permite ajustar el tamaño de lo alto
  • title-permite describir una imagen con el cursor por encima
  • alt-apoyo de busqueda de SEO
  • a-permite agregar un url o link de una pagina, una imagen o una redireccion dentro de la misma pagina
  • target-asigna la opcion de abrir el link en la misma pagina o abriendo otra

notas

Cada una de estas herramientas permite la facilidad de un diseño adecuado para una pagina sabiendo utilizarlas bien, sabiendo su sintaxis o sus reglas basicas sabiendo cuando usar una u otra de las herramientas


Estos programas tienen una carecteristica particular, el modo de sistema en cascada. esto quiere decir que todo programa tiene una forma de orden de avance de descenso

El CSS es el sistema de diseño basico de una pagina el cual se vincula atra ves de un link, una hoja incrustada de etiqueta style o un articulo del mismo tipo style


otro apartado importante el el font-family el cual permite vincular una tipografia para poder editar la tipografia de la pagina y este va mas que nada en el header de la pagina y en las secciones del CSS donde se desea tener dicha tipografia
